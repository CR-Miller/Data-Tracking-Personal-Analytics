# AI Usage Log

## Tools Used
- ChatGPT

## How AI Was Used
AI was used to:
- Debug backend API issues (DELETE, PUT errors)
- Fix Spring Boot controller and service structure
- Help connect frontend (React) to backend
- Assist with troubleshooting errors in terminal
- Generate documentation for Milestone 4

## Ethical Considerations
- All AI-generated code was reviewed and tested before use
- No sensitive or personal data was shared with AI
- AI was used as a support tool, not a replacement for understanding

## Reflection
Using AI significantly improved debugging speed and helped identify issues such as:
- Incorrect file paths
- Missing controller methods
- Backend/frontend connection errors

AI contributed to improving both development efficiency and understanding of full-stack workflows.
