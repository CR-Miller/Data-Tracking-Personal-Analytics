# Peer Testing Results

## Participants
3 peers tested the application.

## Tasks Given
- Add a new task
- View tasks
- Update task due date
- Delete a task

## Results

| Tester | Feedback |
|------|--------|
| User 1 | App was easy to navigate |
| User 2 | Task creation worked smoothly |
| User 3 | Suggested improving visual design |

## Improvements Identified
1. Improve UI styling for better appearance
2. Add clearer labels for inputs
3. Add confirmation for delete action

## Conclusion
Peer testing confirmed that the core features work correctly. Minor UI improvements were identified for better user experience.
